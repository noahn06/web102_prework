# WEB102 Prework - *Sea Monster Web Page*

Submitted by: **Noah Nguyen**

**Sea Monster Web Page** is a website for the company Sea Monster Crowdfunding that displays information about the games they have funded.

Time spent: **12** hours spent in total

## Required Features

The following **required** functionality is completed:

* [x] The introduction section explains the background of the company and how many games remain unfunded.
* [x] The Stats section includes information about the total contributions and dollars raised as well as the top two most funded games.
* [x] The Our Games section initially displays all games funded by Sea Monster Crowdfunding
* [x] The Our Games section has three buttons that allow the user to display only unfunded games, only funded games, or all games.

The following **optional** features are implemented:

* [x] A real-time search bar that allows the user to search for any game that they have heard of!
* [x] Styled web page to include high contrast background as well as cleaner button layout and design

Describe any challenges encountered while building the app.

- Learning about destructuring and the spread operator, and figuring out styling in CSS. It was a big learning curve with lots of referring to documentation.

- Getting used to the process of creating an element, putting data inside the element from the JSON, and then sending that element back into the HTML. I've never done web development like this before.
